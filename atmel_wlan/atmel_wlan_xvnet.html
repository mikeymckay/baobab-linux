NAME

xvnet - ATMEL Wireless Card Monitor Utility for X  

SYNOPSIS

xvnet  

DESCRIPTION

xvnet is the monitor utility for the ATMEL Wireless Card adapters. When the pcmcia module fastvnet_cs or the usb module vnetusba.o is loaded the xvnet application can monitor the device's statistics or change it's runtime parameters.  

NOTE

In order for the application to run the driver has to complete it's initialization. If the device's initialization isn't finished the application won't start.  

Utility Tabs

Monitor Tab.
In this tab the MAC Address, Driver State, Associated Access Point's BSSID, Signal Strength, and Link Quality fields are used only to display the corresponting information. All the other fields can be used for input. In order to change any value you have to Press the Change button, change any value you want to change and then press the Submit value to enable them or the Cancel button to abort your changes.
Statistics
In the Statistics tab you can view info for the management and data frames. By pressing the Clear button you can clear the statistics.
Site Survey.
In this tab you can get a list of nearby access points or ad-hoc stations. After site survey finishes you can select one of those (if you have the necessary information, wep key , wep mode etc) by double clicking the desired access point or station. To initiate a site survey click on the Re-Scan button. If you want to stop an ongoing site survey press the Cancel button. As soon as you select an access point or a station the application will switch automatically to the first tab.
Encryption
In the Encryption tab you can set Encryption settings. 
Encryption Level   : Disabled or 64-Bit,
(in Usb and RFMD boards you can also set 128 bit)
WepKey 1           : value of 1st wep key 
WepKey 2           : value of 2nd wep key 
WepKey 3           : value of 3rd wep key 
WepKey 4           : value of 4th wep key
The Wep Keys length is 10 characters when 64-bit is enabled and 26 characters for 128-bit Encryption Level. 
Wep key to Use     : which wep key is going to be used,
Wep Mode           : Mandatory or Optional,
Authentication Type: Open System or Shared Key

After you have entered the desired settings press the Submit button to enable them.

Advanced (Advanced Card Settings)
Preamble Type          : Long or Short,
Fragmentation Threshold: 256-2346,
RTS/CTS Threshold      : 256-2347,
Version
Driver, Firmware and Application versions.
