NAME

fvnet - ATMEL Wireless Card Console Monitor Utility  

SYNOPSIS

fvnet  

DESCRIPTION

fvnet is the monitor utility for the ATMEL Wireless Card adapters. When the pcmcia module fastvnet_cs or the usb module vnetusba.o is loaded the fvnet application can monitor the device's statistics or change it's runtime parameters.  

NOTE

In order for the application to run the driver has to complete it's initialization. If the device's initialization isn't finished the application won't start.  

Application Screens

Basic Screen
In the first you have 3 options. By pressing c you can configure the running device. By pressing v you can view the current configuration. And finally by pressing x you can leave the utility.
First Screen - View Configuration.
If you press the v key the device's configuration will be displayed on your screen. Here you can see the transmitted and received data packets. The device's state is also displayed. If the device is associated you can see the associated access point's name and it's BSSID. Other information you can see is the channel, the transmit rate, the Operating Mode, the Preamble Type, the Power Management Mode, the RtsCts Threshold and the Fragmentation Threshold. You can also see the encryption settings.
Second Screen - Set Configuration
In the second screen you can configure the running device. By pressing the letter inside the brackets you can set the corresponting value.
